Uzay Simülasyonu – C Konsol Uygulaması

Bu depo, C dili ile geliştirilen konsol tabanlı bir uzay fizik simülasyonu içerir. Program, temel fizik deneylerini Güneş Sistemi’ndeki gezegenlerin farklı yerçekimi ivmelerine göre simüle eder.
 Bu proje bireysel olarak geliştirilmiştir.

 Özellikler 

9 fizik deneyi (serbest düşme, yukarı atış, ağırlık, potansiyel enerji, hidrostatik basınç, kaldırma kuvveti, sarkaç, ip gerilmesi, asansör)

Modüler yapı: her deney ayrı fonksiyon

Gezegen yerçekimi ivmeleri dizi içinde tutulur

Dizi elemanlarına pointer ile erişim

Negatif girişler ternary operatörü ile düzeltilir

Menü tabanlı kullanım, -1 ile çıkış

 Kullanım

Bilim insanı adını gir.

Menüden deney seç.

Gerekli değerleri gir.

Sonuçları tüm gezegenler için görüntüle.

Çıkış: -1

Çalıştırma

Bu proje C dili ile yazılmıştır ve komut satırından derlenip çalıştırılabilir.
